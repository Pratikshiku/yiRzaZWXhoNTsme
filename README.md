## Online Examination Scoring System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/6d33c4110fe04eab8933c0f453aa7c3c/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, administrators.

**Regular User Features:**
1. Login
2. Registration
3. Homepage
4. Exam Rules
5. Exam Paper List
6. Exam News
7. Personal Center
8. Exam Records
9. Wrong Answer Book


**Administrator Features:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. User Management
4. Exam Statistics Management
5. Major List Management
6. Major Type Management
7. Exam Rules Management
8. Forgot Password Personnel Registration Management
9. Exam Scoring Management
10. Exam Paper Management
11. Question Management
12. System Management
13. Exam Management
